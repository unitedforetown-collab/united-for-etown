
# United for E‑Town — Static Site (Netlify)

## Deploy
1. Create a new site in Netlify → **Deploy manually** → drag this folder.
2. In **Site settings → Domain management → Add custom domain**, enter your DreamHost domain.
3. Copy the DNS records Netlify shows you and paste them into DreamHost (Manage Domains → DNS).
4. Wait for DNS to propagate, then reload your site.

## Customize
- Replace form placeholders with Google Forms/Airtable iframes.
- Replace donation placeholder with ActBlue/Stripe.
- Edit pages directly in the .html files.
- Colors are in `styles.css` using CSS variables.

## Assets
Logo and photos are in `/assets/`.
